Binary Search Tree Visualization

INTRODUCTION

A Binary Search Tree (BST) is a special type of binary tree where each node follows a specific ordering property: the value of the left child is smaller than its parent node, and the value of the right child is greater than its parent node.

BSTs are widely used for efficient searching, insertion, and deletion operations. They also form the basis for many advanced data structures like AVL trees, Red-Black trees, and B-Trees.

In this project, we visualize how a BST is constructed from user input and demonstrate tree traversals: Inorder, Preorder, and Postorder.
