A Binary Search Tree is a special type of binary tree that maintains its elements in a sorted manner. It is a non-linear data structure that stores data in the form of nodes, where each node contains a key (or value) and references to left and right child nodes.

A Binary Search Tree is a binary tree in which every node satisfies the following property:

  • The left child (and its entire left subtree) contains values less than the nodeâ€™s value.
  • The right child (and its entire right subtree) contains values greater than the nodeâ€™s value.
  • Both left and right are also BSTs.

This property makes searching, insertion, and deletion operations faster as compared to a normal binary tree.
