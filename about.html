What is a Binary Search Tree?

Definition: A Binary Search Tree (BST) is a node-based data structure where:

  • The left subtree contains nodes with values less than the node.
  • The right subtree contains nodes with values greater than the node.
  • Both subtrees must also be BSTs.

Applications: Efficient searching, insertion, deletion, and as the basis for advanced trees like AVL and Red-Black Trees.

Tree Traversals

// Inorder Traversal (Left → Root → Right)
function inorder(node) {
    if(node != null) {
        inorder(node.left);
        console.log(node.data);
        inorder(node.right);
    }
}

// Preorder Traversal (Root → Left → Right)
function preorder(node) {
    if(node != null) {
        console.log(node.data);
        preorder(node.left);
        preorder(node.right);
    }
}

// Postorder Traversal (Left → Right → Root)
function postorder(node) {
    if(node != null) {
        postorder(node.left);
        postorder(node.right);
        console.log(node.data);
    }
}

Step-by-Step Example

Consider a BST built from inserting: 50, 30, 70, 20, 40, 60, 80

Inorder: 20 → 30 → 40 → 50 → 60 → 70 → 80 (sorted order)

Preorder: 50 → 30 → 20 → 40 → 70 → 60 → 80

Postorder: 20 → 40 → 30 → 60 → 80 → 70 → 50
